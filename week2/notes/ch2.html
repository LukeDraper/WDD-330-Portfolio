Javascript Grammar

  • Javascript uses C-style syntax
  • semi-colons are automatically inserted in Javascript, but it's best to put them in yourself because errors can arise
  • Blocks are sections of code within curly braces
  • whitespace can be used freely

Primitive Data Types

There are 7 data types and 6 of them are primitive data types in Javascript:

  • String
  • Symbol
  • Number
  • Number
  • Boolean
  • Undefined
  • Null
All other data types are objects. Use typeof to determine data type.
Example:
typeof 10
<< 'number'

Assigning and Declaring Variable

While using the const keyword, if it references a non-primitive data type like an array, function, or object then the data inside the object can change. This is known as mutating the object.
Because const variables can't be changed, most variables should be declared as const to make the program more predictable.

Global Scope

Variables declared outsided any block have global scope. They can be accessed from anywhere in the program. Variables with global scope should be kept to a minimum. They can mess things up with same variable names.

Assignment by Reference

Assigning a non-primitive value to a variable is done by reference. Any changes made will affect all references to that object.

Backslash is the escape character in strings. \\ Double backslash is used to place one backslash.

Wrapper Objects

Only objects have properties and methods, so JS makes wrapper objects for primitive data types. This happens in the background.

Template Literals

JS can now use the backtick character ` to deliminate strings. This allows for used of both kinds of quotation marks inside one string without using the backslash. Interpolations can also be used freely inside the backtick characters. It's common to see backticks used for all strings in all ES6 code.

Symbols

Symbols are the only primitives without a literal form. They can create unique values.

Infinity

Infinity is used as an error for when there are numbers too big for the language to deal with. There is also -Infinity.

NaN

Error value for when an operation is attempted, but the result isn't numerical.

Type Coercion

When operations are performed JS tries to make the operands the same type. Sometimes this doesn't work, so it is best to manually change types if needed.
