Initializing Array

Empty array creation:
const heroes = [];
or
const my Array = new Array();
The first version is called an array literal. It's better to use for some reasons, and it requires less typing.

Adding Values to Arrays

Each item of an array can be treated like a variable. You can change the value using the assignment operator =.

heroes[0] = 'Batman';

If you add a value to a later index like:

heroes[5] = 'Aquaman';

The array's size will increase, but the values between the two filled portions will be undefined, and will show up as that if you display the array.

Creating Array Literals

Each value of an array can be a different data type in JS.

Removing Values from Arrays

Use the delete operator to remove an item from an array.

delete avengers[3];
This code will make the element at that index undefined.

Destructuring Arrays

Destructuring an array is taking values out of an array and showing them as individual values. This allows us to assign multiple values at the same time.

const [x, y] = [1,2];
The previous line of code uses an array to assign the values, but Each value exists on its own outside the array when called.

Array Properties and Methods

The length property returns the length of an array:
avengers.length;

arrayName[arrayName.length-1]; Returns the last item in an array.
myArray.length = 8; This line changes the array's length to 8 items. Extra slots are filled with undefined. Shortening an array removes extra elements entirely.

Pop, Push, Shift, and Unshift

  • pop(); method removes the lasts item of the array and return its values.
  • shift() method works like pop() but removes the first item of the array.
  • push() method appends the value in the parentheses to the end of the array.
  • unshift() works like push but at the beginning of the array.

Merging Arrays

  • concat() method
    • concat() method merges an array with one or more arrays.
    • Form: myArray.concat(array2)
    • Second array is appended to the first.
  • spread operator (...)
    • The spread operator works like concat, but uses three elipses acting as the spread operator rather than a method.
    • Example: avengers = [ ...avengers, ...['Hulk','Hawkeye', 'Black Widow'] ];
      Result: ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']

Join method

The join() method turns an array into a string with each item in the array being separated by a comma. You can also choose a different separator.

Slicing and Splicing

  • slice() method
    • Slice selects part of an array and return that part of it without removing it from the original.
    • Example: avengers.slice(2,4) // starts at the third item (index of 2) and finishes at the fourth (the item with index 4 is not included)
      Result: ['Thor', 'Hulk']
  • splice() method
    • myArray.splice(starting index, number of items, values at this point and after will replace items removed from array );
    • Using splice with a 2nd value of zero works to insert an item into the array.
    • splice returns the values that were removed in the form of an array.

Reverse

  • Reverses order of array within the variable.

Sort

  • Sort the array permanently.
  • Can be undone with a callback.

Finding if a value is in an array

  • indexOf() method
    • Returns the index of the first occurence of a given value in an array
    • Example: myArray.indexOf(value to check for)
    • Return -1 if the search value isn't in the array.
  • includes() method
    • myArray.includes(Value to search for, optional argument for starting index);
    • Returns a boolean value that is true if the search value occurs in the array at least once.

Multidimensional Arrays

  • Multidimensional Arrays can be used to create a coordinate system.
  • Accessed using two sets of indeces inside square brackets: multiArray[0][3];
  • The spread operator can be used to flatten a multidimensional array into a single dimension.

Sets

Data structure that represents a collection of unique values. No duplicates. No need to check for duplicates. They aren't allowed. It's quick to find if a particular value is in the set.

Creating a new set:
const list = new Set();

Adding a new set:
list.add(1);

Duplicate variables that are added to the set are ignored instead. However, type coercion is not used on values being added to a set, so the same value can be added as a different data type.

Set Methods

  • size() method returns the number of elements in a set.
  • has() method checks if a value is in a set, and is much faster than using includes() or indexOf().
  • You can't use index notation to look at individual values in a set.

Removing Values from sets

  • delet() removes a value from a set, and returns true if the variable if removed. It returns false if the value isn't in the set.
  • clear() removes all values from the set

    • Converting Sets to Arrays

    A set placed into an array literal along with the spread operator will be converted into an array.

    You can also use the Array.from() method.
    const shoppingSet = new Set().add('Apples').add('Bananas').add('Beans');
    const shoppingArray = Array.from(shoppingSet);

    Weak Sets and Avoiding Memory Leaks

    Objects added to sets won't be garbage-collected, and this can cause memory leaks. You can declare a weak set to avoid this:
    const weak = new WeakSet();
    Only non-primitive data types can be added to weak sets. Adding primitive values will throw a type error. Weak sets don't have access to the values contained in arrays in the weak set, so returning the contents of a weak set does not show that it contains the array, but you can check using the has() method.

    Maps

    • Maps keep a list of key-values pairs.
    • Maps can use any data type as a key whereas Objects can only use strings.
    • There is only one way to create maps:
      const romanNumerals = new Map();
    • The set() method can be used to add a key and value pair to a map. The first value is the key and the second is the value:
      romanNumerals.set(1,'I');

    Ternary Operator (shorthand for writing an if else statement)

    ? is the ternary operator. It takes three operands:
    condition ? (//code to run if condition is true) : (//code to run if condition is false)

    Example:
    const n = 5;
    n%2 === 0 ? console.log('n is an even number') : console.log('n is an odd number');
    Output: 'n is an odd number'
    Shortened example within a template string:
    console.log(`n is a ${(n%2 === 0)? 'even' : 'odd'} number`);
