Object Method Notes

Functions that are properties of objects are object methods. These allow them to do actions that are related to the objects.

Method shorthand

Orginal: 

                user = {
                    sayHi: function() {
                        alert("Hello");
                    }
                };
Method shorthand(functions the same): 
                user = {
                    sayHi() {
                        alert("Hello");
                    }
                };

"This" keyword in methods

The keyword "this" can be used to reference the object used to call the method. It can be used to acccess object variables. If there is a member variable called name then it can be referred to using this.name

"this" is not bound in javacript

When obj.f is called then this is obj during the call of f.

If this is called without an object then it is undefined. "this" can also refer to the global object, which could be a window in a browser.

The value of "this" ies evaluated at call-time.

Arrow function have no "this"

If "this" is called from within an arrow function then it will use "this" from the outer function. Example: 

                let user = {
                    firstName: "Ilya",
                    sayHi() {
                      let arrow = () => alert(this.firstName);
                      arrow();
                    }
                  };
                  
                  user.sayHi(); // Ilya
